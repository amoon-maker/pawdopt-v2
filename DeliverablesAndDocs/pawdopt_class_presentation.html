02 / 08

The problem we are solving

Pet adoption in Quebec is fragmented, unverified, and unsafe
The gap in the market
No centralized platform in Quebec for pet adoption and rehoming. Adopters rely on Facebook posts, outdated shelter websites, and unstructured emails. Scams and unverified listings are common.
Our target client
Animal shelters — organizations like SPCA Montreal — that need a modern digital platform to list verified animals, screen adopters, and manage the full rehoming workflow.
What Pawdopt v2 delivers to the shelter client
A branded web platform where every listing is document-verified before going live, adopters go through a structured application wizard, and all requests are tracked in one dashboard — replacing fragmented email and social media workflows entirely.
03 / 08

What we are building

Four access levels — one unified platform
Guest
Browse pet listings and care guides freely — no account needed. Registration only required to take action.
Adopter
Register, apply through a 5-step wizard, track applications, save favourites, and message rehomers.
Rehomer
List pets through an 8-step wizard with mandatory vaccination document upload. Review and accept applicants.
Admin
Review uploaded documents, approve or reject listings before they go live, manage all users and analytics.
C# ASP.NET Core MVC SQL Server Entity Framework Core GitHub (shared repo) EN/FR toggle Dark / light mode

05 / 08

Our 8 user stories for Deliverable I

Covering both the Adopt side (Amina) and Rehome side (Isabela)
US-1
Guest — browse pet listings without an account
US-2
New user — register with role selection (Adopter or Rehomer)
US-3
New user — verify email with 6-digit code before accessing the platform
US-4
Registered user — log in and be redirected to the correct role dashboard
US-5
Logged-in user — role-based dashboard separation (Adopter / Rehomer / Admin)
US-6
Adopter — filter pets by type, breed, age, size, and location
US-7
Adopter — view full pet detail page with story, health badges, and Apply button
US-8
Rehomer — complete rehome wizard steps 1–3 with progress saved between steps
US-9
Rehomer — upload vaccination document before listing is submitted for admin review
